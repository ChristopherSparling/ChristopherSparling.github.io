Big 5 Gender Predictor

Repo

Built With: R, RStudio, tidyverse, assorted packages

   Project presented as R markdown notebook which draws its data from two associated .csv files. The report filters, cleans, and transforms the dataset (pulled from the Open Source Psychometrics Project) into a usable form. The data is split into train/test sets and fed into an assortment of machine learning algorithms including MDA, Neural Networks, and kNN. These generate models to predict the gender feature of the respondants which are then tested for basic ML metrics. Users can answer the quiz used to collect the data themselves, insert the data, and receive back

Planned Feature(s): Graphical visualization of personality using percentile scores, further exploratory analysis of dataset

MMA Combo Practice Tool

Repo

Built With: Python, Pygame

   A fairly straightforward program, but one that I actively use and enjoyed making, the Combo Practice Tool generates random sets of combos from pre-defined options that take approximately one round (3 minutes) to execute. The tool calls out the moves comprising the current combo and displays them to the user. The program is highly configurable, allowing you to change round length, the time allotted to each combo, and even add in kicks and switch-kicks at the end of the combo for a more challenging set. For my own training I have added combos developed by my gym's coach, but have excluded these from the repository in good faith

Planned Feature(s): GUI/functionality to allow user to pause during combos and display moves in larger font, improve ease of adding new combos which requires manually downloading audio files currently
